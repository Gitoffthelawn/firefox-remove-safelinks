Test page for the "Disable Microsoft Safe Links" addon

Below, you'll find links that look like the ones that are "protected" by Microsoft Safe Link.

How to debug

Use Firefox' debugging feature for add-ons to verify that the links were indeed detected and blocked.

  1. Visit about:debugging#/runtime/this-firefox
  2. Click "Load Temporary Add-on..."
  3. Select the manifest.json file this repository
  4. Click the "Inspect" button
  5. In the Developer Tools window, click on "Console"

When a Safe Link redirect was blocked, a debug message is logged to the console.
